Coinsの模範解答(python)

kaisuu500 = int(input()) kaisuu100 = int(input()) kaisuu50 = int(input()) kingaku = int(input()) count = 0 for a in range(kaisuu500+1): for b in range(kaisuu100+1): for c in range(kaisuu50+1): if 500 * a + 100 * b + 50 * c ==kingaku: count += 1 print(count)

Card Game for Twoの模範解答(python)

Alice = 0 Bob =0 N = int(input()) a = list(map(int,input().split())) a = sorted(a, reverse=True) for i in range(N): if i % 2 == 0: Alice = Alice + a[i] else: Bob = Bob + a[i] c = Alice - Bob print(c)
