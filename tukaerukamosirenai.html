ゾロ目判別プログラミンング(サイコロは３つ)

問題文3つのサイコロを振ってゾロ目が出た場合は'zorome'そうでない場合は'zoromejanai'を表示しなさい
import random←randomモジュールをインポートする

dice1 = random.randint(1, 6)←１から６までの数字を生成(サイコロ1)
dice2 = random.randint(1, 6)←１から６までの数字を生成(サイコロ2)
dice3 = random.randint(1, 6)←１から６までの数字を生成(サイコロ3)
if dice1　== dice2 == dice3:←サイコロ１,２,３の値が同じ場合
←→print("zorome")←zoromeと表示する
else:←サイコロ１,２,３の値が違う場合
←→print("zoromejanai")←zoromejanaiと表示する
